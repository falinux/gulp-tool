## Getting started with gulp-tool

### Prerequisites

	Linux
	Docker
	Git

### Download and install	

	$ git clone https://github.com/falinux/gulp-tool.git ~/gulp-tool
	$ cd ~/gulp-tool
	$ ./install.sh
	
### Enter into gulp-tool

	$ cd [working directory]
	$ gulp-tool
	
### Updage gulp-tool
	
	$ cd ~/gulp-tool
	$ gulp-tool
	$ ./gt-update
